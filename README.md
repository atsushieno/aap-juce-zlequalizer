# aap-juce-zlequalizer

It is a port of [ZL-Audio/ZLEqualizer](https://github.com/ZL-Audio/ZLEqualizer) to [AAP (Audio Plugins For Android)](https://github.com/atsushieno/aap-core) using [aap-juce](https://github.com/atsushieno/aap-juce).
